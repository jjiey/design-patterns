实际上，设计模式要干的事情就是解耦。
创建型模式是将创建和使用代码解耦
结构型模式是将不同功能代码解耦
行为型模式是将不同的行为代码解耦
    具体到观察者模式，它是将观察者和被观察者代码解耦。
